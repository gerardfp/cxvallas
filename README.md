# cxvallas

## Option1: Run local
```
git clone https://github.com/gerardfp/cxvallas
```

```
node server.js
```

## Option2: build docker image
```
docker build -t cxvallas .
```

``` 
docker run -dp 8888:8888 cxvallas
```

## Option3: run from docker hub
``` 
docker run -dp 8888:8888 gerardfp/cxvallas
```
